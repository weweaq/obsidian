1.安装：yum -y install nginx

2.服务名：nginx

重启squid服务 ：systemctl restart nginx

查看服务状态 ：systemctl status nginx

开机自启：systemctl enable nginx

3.配置目录：cd /etc/nginx

4.主配置文件：vim /etc/nginx/nginx.conf

5.详细配置文件：vim /etc/nginx/conf.d
